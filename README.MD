## Project setup

```
Install Mongo Community Edition : https://docs.mongodb.com/v3.2/tutorial/install-mongodb-on-ubuntu/
```
```
Install Nodemon : sudo npm install -g nodemon
```
```
Use: service mongod start + nodemon to run Appplication
```
